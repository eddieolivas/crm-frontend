# crm-frontend

The frontend to a demo MERN stack CRM ticket system built with React, Redux, Redux Toolkit, React-Bootstrap, and Axios. 

![alt text](https://www.chrysaliswebdevelopment.com/wp-content/uploads/2021/11/Screen-Shot-2021-11-27-at-4.02.52-PM.png)

The app is running from a Docker container on AWS using ECR, ECS, and EC2.
You can create a new user and you will receive an activation email. Once activated, you can log in/log out and create/update tickets.
